Twim
====
This is a fork of [Twim](http://code.google.com/p/twim/) by [Tommi Laukkanen](http://www.substanceofcode.com).

Enhancements
------------
* can be built with ant from the CLI (no need for Netbeans)
* includes the *Bouncy Castle* crypto libraries
